# 🌌 NexusAI - Next-Gen Productivity Interface

NexusAI is a high-converting, modern landing page built with **React** and **Tailwind CSS**. It features a dark-themed UI with glassmorphism effects, neon accents, and smooth animations, designed specifically for an AI Productivity Tool.

---

## 🚀 Key Features

* **Glassmorphism UI:** Modern frosted-glass navigation and card designs.
* **Hero Section:** Catchy tagline with high-performance gradient text.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
* **Modern Iconography:** Integration with Lucide-React for clean visuals.
* **Pricing Architecture:** Clean, high-converting subscription tiers.
* **Performance:** Built with Vite for ultra-fast load times.

---

## 🛠️ Tech Stack

* **Frontend:** [React.js](https://reactjs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Fonts:** Inter & Space Grotesk (Google Fonts)

## ⚙️ Installation & Setup


1. **Clone the repository:**
```bash
git clone [https://github.com/your-username/nexus-ai.git](https://github.com/your-username/nexus-ai.git)
cd nexus-ai

```


2. **Install dependencies:**
```bash
npm install

```


3. **Run the development server:**
```bash
npm run dev

```


4. **Build for production:**
```bash
npm run build

```
